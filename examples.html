A sample swbisdefaults file

Verbosely install an RPM to a remote host where neither rpm nor swbis is installed:

        # swinstall -x verbose=8 --allow-rpm -s /tmp/gzip-1.3.3-5.i386.rpm @low04:/mnt/test 1>OUTPUT 2>&am;1

Unpack a portable archive on the remote host host1 as user1

	# swcopy --no-audit  @user1@host1:/var/tmp/newdir/ <archive.tgz

Copy a directory to another host behind a firewall

	# swcopy -Wno-audit -s /var/tmp/test-root/ @root@firewall@@192.168.1.1:/var/tmp/

Copy a file FILE to the home directory on host1 and host4, and host2 where there is no permission.

	# swcopy -Wno-audit -x verbose=5 -s `pwd`/FILE @ host1 host2:/ host4 1>OUTPUT 2>&1

A binary package with control directories:

	# swpackage -Wdir=swbis-1.00 -s PSF @- | tar tvf -

A binary package with a single fileset and zero length control directories:

	# swpackage -Wdigests -s PSF @- | tar tf -

A binary package made from uninstalled source using directory mapping:

	# swpackage -s PSF @- | tar tf -

Install a posix package on stdin at "/" on the localhost:

	# swinstall -s -

Install a posix package on stdin to /mnt/test on host1:

	# swinstall -s - @ host1:/mnt/test

Use the preview option of swinstall:

	# swinstall -s /tmp/gzip-1.3.3-5.tar.gz -p 1>OUTPUT

Use the preview option of swinstall:

	# swpackage --unrpm < gzip-1.3.3-5.i386.rpm | swinstall -p 1>OUTPUT

Create a signed source package:


	# swpackage -s PSF -Wdir=swbis-10.1 -Wgpg-name="Test User" -Wsign  @- | tar tf -

Output from signature verification


	# swpackage -s PSF -Wdir=swbis-10.1 -Wgpg-name="Test User" -Wsign  @- | swverify --checksig

View a binary RPM in tar format.


	# swcopy --unrpm <gzip-1.3.3-5.i386.rpm | tar tvf -

View a source RPM in tar format.


	# swpackage --archive-digests --digests --2posixformat <rpm2html-1.7-8.src.rpm | tar tvf -

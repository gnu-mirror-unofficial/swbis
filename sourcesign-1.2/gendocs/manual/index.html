This manual (sourcesign) is available in the following formats:

  • HTML (44K characters) - entirely on one web page.
  • HTML - with one web page per node.
  • HTML compressed (12K gzipped characters) - entirely on one web page.
  • HTML compressed (16K gzipped tar file) - with one web page per node.
  • Info document (12K characters gzipped tar file).
  • ASCII text (28K characters).
  • ASCII text compressed (12K gzipped characters).
  • TeX dvi file (16K characters gzipped).
  • PostScript file (68K characters gzipped).
  • PDF file (188K characters).
  • Texinfo source (12K characters gzipped tar file)
